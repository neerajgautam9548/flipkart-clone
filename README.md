# 🛒 Flipkart Clone

A modern and responsive **Flipkart Clone** built using **HTML, CSS, and JavaScript**, designed to replicate the look and feel of India's leading e-commerce platform. This project focuses on creating a visually appealing user interface, smooth user experience, and responsive design principles.
